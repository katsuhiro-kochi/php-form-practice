## 概要

COACHTECH 教材 Tutorial 7-4「フォームとデータ受け渡し ハンズオン演習」で作成した成果物です。
ユーザーの名前、年齢、メールアドレスを登録するもので、入力画面でユーザーから受け取り、確認画面に移動して登録情報を表示し、入力画面に戻るか登録に進むかの意思決定をユーザに委ね、ユーザー登録するというもの

## 使用技術

- PHP 8.x
- HTML5（フォーム要素）
  （**他に使ったものがあれば追記してください**）

## 学んだこと

- ファイル間の移動（スーパーグローバル変数の別ファイルへの引き継ぎ（inputのtypeの要素に"hidden"を使えば、スーパーグローバル変数を引き継げる））
- HTMLフォームから PHP に値を渡す方法（POST,GET,スーパーグローバル変数での受け取りなど）
- htmlspecialchars関数の必要性（スーパーグローバル変数で受け取り、xss攻撃されないように、htmlspecialchars関数を使い想定しない文字などを修正できること）
